# Editor Workflow (Michel)

## Login

1. Go to `/studio`.
2. Sign in with invited Sanity account.

## Update landing page

1. Open `Landing Page`.
2. Edit intro text.
3. Add/reorder/remove hero images.
4. Publish.

## Add or update paintings

1. Open `Paintings`.
2. Create new painting or edit existing one.
3. Fill title, year, status, image, caption.
4. Publish.

`Available` page updates automatically from paintings where `status = available`.

## Update commissions

1. Open `Commissions Page`.
2. Edit intro text.
3. Update 4 process steps (title, text, image).
4. Update recent examples.
5. Publish.

## Update happy clients

1. Open `Happy Clients`.
2. Add/remove/reorder photos.
3. Add optional caption.
4. Publish.

## Publish timing

- Local: immediate
- Production: updates appear after webhook revalidation and cache refresh (usually under 60 seconds)
