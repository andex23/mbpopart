# MB Pop Art Documentation

This documentation covers the site architecture, CMS setup, publishing flow, and deployment.

## What this project does

- Keeps layout/templates controlled in code.
- Moves all editable text and media content to Sanity.
- Allows non-technical editing through Sanity Studio.

## Core principles

- Content is editable.
- Layout structure is fixed.
- Navigation labels/order/visibility are editable, but route keys remain fixed.
- Image rendering must keep ratios and avoid distortion.
